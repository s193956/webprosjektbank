Viktig;
Ved innlogging tast inn fødselsnummeret først, 12345678912, deretter trykker man på få ny engangskode.
Deretter blir man bedt om å taste fødselsnummer nedenfor, 12345678912, deretter igjen engangskoden som man får, så til slutt det personlige passordet som er "123", dette er lagret som et hash men man kan se det i LoggInnControlleren.
